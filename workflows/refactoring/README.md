# Refactoring Workflows

Workflows for code modernization and cleanup.

## Workflows
- [Modernize Code](modernize-code.mwc.yaml): Refactor legacy JS/TS to modern standards.
- [Performance Audit](performance-audit.mwc.yaml): Analyze code for performance bottlenecks and memory leaks.
- [React Hooks Converter](react-hooks-converter.mwc.yaml): Migrate React class components to functional components using Hooks.
