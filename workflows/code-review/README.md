# Code Review Workflows

Workflows designed to automate and enhance the code review process.

## Workflows
- [Standard Review](standard-review.mwc.yaml): A comprehensive code review covering logic, style, and security.
- [Bug Fixer](bug-fixer.mwc.yaml): Analyze an error message and suggest a code fix.
- [Accessibility Review](accessibility-review.mwc.yaml): Review UI code for ARIA standards and accessibility best practices.
- [Secret Exposure Check](secret-exposure-check.mwc.yaml): Scan code for hardcoded secrets, API keys, or sensitive credentials.
- `security-audit.mwc.yaml`: Focused security audit of code changes.
