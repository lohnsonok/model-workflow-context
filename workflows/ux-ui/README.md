# UX/UI Workflows
Workflows for design research, user feedback, and UI audits.

## Workflows
- [User Interview Script](user-interview-script.mwc.yaml): Generate non-biased discovery scripts.
- [UX Heuristic Audit](ux-heuristic-audit.mwc.yaml): Perform a UX audit based on Nielsen's heuristics.
- [UI Feedback Summarizer](ui-feedback-sum.mwc.yaml): Summarize raw user feedback into actionable design changes.
